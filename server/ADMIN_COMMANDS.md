# Quick Admin Commands Reference

## Most Common Commands

### Promote Admin to Super Admin
```bash
cd server
npx tsx src/scripts/promote-super-admin.ts your-email@example.com
```

### Reset Admin Password
```bash
cd server
npx tsx src/scripts/reset-admin-password.ts your-email@example.com NewPassword123
```
